## What This Is

Token Optimization for Context Engineers. 4.8 KB WASM. Sub-millisecond. Zero dependencies.

## Stack

- **Language:** TypeScript

## Context

- **Who:** Context Engineers making LLM API calls (Claude, Grok, Gemini, GPT)
- **What:** Pre-call token checks. Know the cost before the call leaves your machine. Skip, reduce, or route.
- **Why:** Most apps have no idea what a call costs until the invoice arrives. Slash tells you before the request hits the wire.
- **Where:** npm (slash-tokens), slashtokens.com, mcpaas.live/slash
- **When:** v1.1.0, April 2026
- **How:** npm install slash-tokens / bunx slash-tokens
